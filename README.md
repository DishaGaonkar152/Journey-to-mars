# Journey-to-mars
An immersive interactive web experience that simulates a complete journey to Mars — from launch and space travel to landing and exploration — powered by GSAP, Canvas, and modern UI animat

## 📁 Project Structure
-Journey-to-Mars.html

# Features
🚀 Launch Simulation Realistic rocket launch sequence with animated stages and mission status.

🌠 Space Travel Experience Dynamic starfield using Canvas and real-time journey progress tracking.

📊 Interactive Mission Timeline Slider-based timeline showing key milestones from Earth to Mars.

🛬 Landing Simulation Animated lander descent with telemetry data (altitude, velocity, accuracy).

🗺️ Mars Exploration Module Interactive map with clickable points of interest and rover movement.

🎯 Progress Tracking Visual indicators for mission completion and exploration status.

✨ Advanced UI/UX Smooth animations powered by GSAP and ScrollTrigger.
